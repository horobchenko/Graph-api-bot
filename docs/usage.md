# Usage Guide

## Start the bot
Send `/start` to the bot in Telegram.

## Ask questions
The bot retrieves relevant documents from ChromaDB and uses OpenAI to answer questions about:
- property databases
- regulations
- contract templates
- real estate policies

## Expected workflow
1. Configure Graph API access.
2. Download or index document sources.
3. Ask the bot a question.
4. Review the answer grounded in retrieved context.
